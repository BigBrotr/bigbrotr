# `bigbrotr/monitoring/grafana/provisioning`

Auto-provisioned Grafana configuration for the `bigbrotr` deployment.

## What Lives Here

- [`dashboards/`](dashboards/README.md): provisioned dashboard manifests and
  JSON definitions.
- [`datasources/`](datasources/README.md): provisioned datasource definitions.

## Rules

- Keep provisioning assets aligned with the shipped monitoring topology and the
  metric/query contracts they visualize.
