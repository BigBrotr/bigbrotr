# `guides`

Deep-dive engineering references consumed by the project-local command set.

## What Lives Here

- `architecture-extensibility.md`: service, NIP, and deployment extension
  guidance.
- `domain-model.md`: domain invariants and model rules.
- `git-workflow.md`: branch, commit, PR, and release discipline.
- `postgresql.md`: schema, query, and database conventions.
- `python-async.md`: async service and concurrency patterns.
- `testing.md`: test structure, fixtures, mocking, and coverage expectations.

## Rules

- Keep these guides implementation-honest and consistent with `AGENTS.md`,
  `README.md`, and the live codebase.
- Use this folder for reusable deep references; short task procedures belong in
  `../commands/`.
