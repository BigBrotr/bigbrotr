# images

Brand image assets referenced directly by the MkDocs site configuration.

## What Lives Here

- `logo.webp`: header logo used by the Material theme.
- `favicon.png`: site favicon referenced by `mkdocs.yml`.

## Rules

- Keep filenames stable unless `mkdocs.yml` is updated in the same change.
- Replace these assets intentionally; they are part of the public docs brand
  surface, not disposable placeholders.
