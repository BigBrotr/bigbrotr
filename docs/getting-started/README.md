# getting-started

Public onboarding pages for first contact with BigBrotr.

## What Lives Here

- `index.md`: landing page for the onboarding path.
- `installation.md`: prerequisites and installation options.
- `quickstart.md`: shortest path from clone to a working local runtime.
- `first-deployment.md`: first real deployment path using the built-in stack.

## Rules

- Keep this folder focused on the initial operator or contributor ramp-up.
- Promote longer operational material to `docs/how-to/` or `docs/guides/`
  instead of expanding this section indefinitely.
