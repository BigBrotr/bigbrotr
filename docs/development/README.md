# development

Contributor-facing engineering pages shipped in the MkDocs navigation.

## What Lives Here

- `index.md`: landing page for contributor and maintainer workflows.
- `setup.md`: local environment setup, prerequisites, and verification.
- `testing.md`: test layout, fixture conventions, and execution commands.
- `coding-standards.md`: linting, typing, import, and documentation rules.
- `sql-templates.md`: SQL generation system, storage profiles, and override
  flow.
- `contributing.md`: branch, commit, review, and pull request expectations.

## Rules

- Keep `index.md` aligned with the actual page set exposed in `mkdocs.yml`.
- Add new contributor-facing pages here only when they belong in the public
  Development nav.
